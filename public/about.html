Overview

This website is to help students at Trinity College navigate the past examination papers more efficiently.

How to use

Find modules that have past examination papers you want quicker access to and add them to your-modules by ticking the box beside them. When you visit the home-page in future, the modules you have ticked will be shown there and you can get to past papers in less clicks.

We hope you find the website helpful!

Problems?

Links are put into modules via their module code. If a paper isn't in your module, it may have a different module code on TCD's website. Search for the right code in the search box to add it to the site.

Also if your module isn't in your course on the 'All courses' section, add it to your modules using the search box and its module code.

If you have bugs to report, or have any general feedback, please contact nating@tcd.ie
